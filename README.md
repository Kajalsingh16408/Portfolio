# 🌐 Portfolio Website

A simple and responsive portfolio website created using **HTML, CSS, and JavaScript**.  
It highlights my projects, skills, and contact information in a clean and recruiter-friendly way.

---

## 🚀 Features
- Responsive layout with pure CSS (no frameworks)
- Smooth navigation using vanilla JavaScript
- Sections: **Home**, **About**, **Projects**, **Contact**
- Lightweight and fast loading

---

## 🛠️ Tech Stack
- **HTML5** – structure and content
- **CSS3** – styling and responsiveness
- **JavaScript (ES6)** – interactivity and DOM manipulation
